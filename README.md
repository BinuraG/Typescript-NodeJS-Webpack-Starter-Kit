> Requires **Node v8.9+**

#### To run:

```
$ npm start
```

# Features

* Write NodeJS with Typescript + ES7
* Dockerfile for containerization
* ExpressJS installed out of the box
* Webpack Typescript tranpilation to es6 (Supported by Node 8.9 and later)
